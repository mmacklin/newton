5. Method Details

Alpha Sweep

Convergence curves for different step lengths (under-relaxation parameter α). Full GS step (α=1.0) overshoots; smaller values prevent overshoot but may converge too slowly.

Jacobi vs Gauss-Seidel

Jacobi updates all vertices simultaneously (no sequential color-group updates). This eliminates cross-color interference but loses GS information propagation. (Easy regime only.)

6. Methodology

Test Setup

  • Mesh: Unisex t-shirt (~6400 vertices, ~12700 triangles)
  • Material: Neo-Hookean membrane + dihedral bending
  • Self-contact: Enabled for all tests
  • Seeds: 4 random initial configurations (randomized drop height, rotation, lateral offset)
  • Simulation: 60 FPS, 10 substeps/frame, 10 VBD iterations/substep

Trajectory-Based Convergence Test

A reference trajectory is recorded under baseline GS. At sampled substeps, (positions, velocities) are saved. Each method is then evaluated from these identical starting states by running a single substep with convergence tracking. This eliminates the confound of compounded inter-substep drift.

Full Rollout Test

Each method runs an independent simulation. After each frame, a probe substep measures the force residual at the current state. This shows whether per-substep convergence/divergence compounds over a full trajectory.

Metric

RMS Force Residual ||∇G(x)|| = RMS of per-vertex total force (inertia + elastic + bending + contact). G(x) is the implicit Euler variational energy; at the exact solution ∇G = 0. This metric is independent of step size / relaxation strategy.
